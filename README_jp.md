# VoxSlice

[English](README.md)

**Live demo: [https://yokazek.github.io/vox-slice/](https://yokazek.github.io/vox-slice/)**


**ブラウザで完結する、高精度な音声スライス体験。**

VoxSliceは、WAVやMP3ファイルを直感的に分割できる高速なウェブツールです。波形エディターと無音自動検出により、インストール不要で精密なスライスと一括書き出しが可能です。

## 主な機能

- **直感的な波形編集:** ドラッグ＆ドロップ対応、スムーズなズーム、精密なシーク操作。
- **スマート無音検出:** 音量としきい値を指定して、自動的に分割ポイントを検出。
- **非破壊編集:** 書き出し前に各セグメントのプレビューや有効/無効の切り替えが可能。
- **一括書き出し:** すべてのセグメントをZIP形式で一括ダウンロード、または個別に保存（WAV/MP3対応）。
- **プロジェクト保存:** 編集状態をJSONファイルとして保存し、いつでも再開可能。
- **プライバシー保護:** すべての処理はブラウザ内でローカルに行われます。音声データがサーバーに送信されることはありません。

## 使い方

1. **アップロード:** WAVまたはMP3ファイルをドラッグ＆ドロップします。
2. **分割:** 波形をダブルクリックするか、**Enterキー**を押して分割ポイントを追加します。**自動無音検出**機能も利用できます。
3. **調整:** 右クリックまたはリストのトグルボタンで、各区間の有効/無効を切り替えます。
4. **書き出し:** フォーマット（WAV/MP3）を選択し、一括または個別にダウンロードします。

## ショートカットキー

- **Space:** 再生 / 一時停止
- **Enter:** 現在位置で分割
- **Ctrl + Z / Y:** 元に戻す / やり直し
- **矢印キー / Tab:** 前後の分割ポイントへ移動
- **ダブルクリック:** 波形上で直接分割
- **右クリック:** 区間の有効 / 除外を切り替え
- **ホイール回転:** 波形の拡大 / 縮小

## 使用技術

- Vanilla JavaScript (ES Modules)
- Web Audio API
- [WaveSurfer.js](https://wavesurfer-js.org/) (波形表示)
- [lamejs](https://github.com/zhuk/lamejs) (MP3エンコード)
- [JSZip](https://stuk.github.io/jszip/) (ZIP圧縮)
